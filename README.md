# SubNotifier
Simple Sub and Resub notifier for Twitch

Questions? Open an issue or tweet [@d0p3t](https://twitter.com/d0p3t)


## Requirements
1. Node
2. NPM

## Installation
1. `npm install`
2. Edit config.js

## Usage
Run `node app.js`

* Hide chat? Set `debug: false` in `config.js`.  
* Create your own custom messages? Edit `app.js`
